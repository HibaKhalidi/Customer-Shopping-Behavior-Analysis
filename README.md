# Analyse du Comportement des Clients

Projet d'analyse du comportement d'achat des clients pour comprendre les habitudes, identifier les clients à forte valeur et analyser les tendances saisonnières. Les données sont nettoyées, traitées et chargées dans une base PostgreSQL pour générer des insights décisionnels.

## Technologies
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
- PostgreSQL + pgAdmin
- SQLAlchemy & psycopg2

## Fichiers Principaux
- `Customer_Shopping_Behavior_Analysis.ipynb` – Analyse complète
- Scripts SQL/ETL pour le chargement en base
- Dataset : comportement d'achat des clients

## Fonctionnalités Clés
- Analyse exploratoire des données (EDA)
- Visualisations : histogrammes, bar charts, heatmaps
- Nettoyage et encodage des données
- Export vers des tables SQL
- Insights sur les comportements d'achat
